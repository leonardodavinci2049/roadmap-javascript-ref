#! JavaScript Shebang Um Shebang é um sinal numérico e um ponto de exclamação (#!) no início de um script: #!/usr/bin/env node O exemplo acima informa ao sistema operacional para usar o programa node para executar o script. Agora, você pode executar código JavaScript com ./fileName.jsem vez de node fileName.js. #! também é chamado de exclamação aguda, hashbang, pound-bang ou hash-pling.
